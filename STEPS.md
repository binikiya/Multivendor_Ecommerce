## Multivendor Ecommerce Website
=========================================
Setup Projects and Creating Database Models
-------------------------------
1. Python
2. Install Virtual Environment - pip install virtualenv
3. Create Virtual Environment - python -m venv env
4. Activate Virtual Environment - .\env\Scripts\activate
5. Install Django - pip install django
6. Create Django Projects - django-admin startproject backend
7. Run Django Project - python manage.py runserver
8. Install postgreSQL
9. Install pgAdmin4
10. Install psycopg2-binary - pip install psycopg2-binary
11. Install DjangoRestFramework - pip install djangorestframework
